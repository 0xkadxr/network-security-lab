# Networking Labs

Hands-on Cisco IOS lab exercises covering fundamental networking concepts. Each lab includes topology diagrams, step-by-step configuration instructions, and verification commands.

## Lab Index

| Lab | Topic | Difficulty |
|-----|-------|------------|
| [Lab 01](lab01-basic-config/) | Basic Router/Switch Configuration | Beginner |
| [Lab 02](lab02-vlan-config/) | VLAN Configuration & Inter-VLAN Routing | Intermediate |
| [Lab 03](lab03-acl-firewall/) | Access Control Lists & Firewall Rules | Intermediate |
| [Lab 04](lab04-qos/) | QoS - Traffic Classification & Queuing | Advanced |
| [Lab 05](lab05-vpn-ipsec/) | Site-to-Site VPN with IPSec | Advanced |

## Requirements

- Cisco Packet Tracer 8.x or GNS3
- Basic understanding of the OSI model and TCP/IP
- Familiarity with the Cisco IOS CLI

## How to Use

1. Read the lab README for objectives and topology
2. Build the topology in Packet Tracer or GNS3
3. Apply the configurations from `config.md`
4. Follow the verification steps to confirm correct operation
5. Use the troubleshooting section if something does not work
